Additional reports and Additional data processors lists allow you to run custom reports or data processors.

To run an additional report or data processor

  • Select the report or data processor and then click Execute.
     

To add a report or data processor to the list

  • When adding the report or data processor to the application as described in Additional reports and data processors, select the default list section of the command interface.
     

To hide or show additional reports or data processors in the list

  • Click Customize the list of my reports or Customize the list of my data processors, add or remove items, and then click OK.
