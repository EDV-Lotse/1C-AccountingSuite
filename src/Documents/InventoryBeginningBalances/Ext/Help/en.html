Inventory beginning balances

1. Use the Inventory Beginning Balances records in the Accounting workspace.

2. Select the Item, Location (if the Multi Location option is enabled), and enter quantity and total value for this inventory item.

3. When a beginning balance record is saved the system will create a record in the Inventory Journal on the Start Date.

4. The Inventory Beginning Balances don't perform Inventory Revaluation which is done by inventory inflow our outflow transactions (such as Purchase Invoices or Sales Invoices).

To ensure accurate inventory costing information inventory beginning balances need to be entered before any inventory transactions. An alternative (less feasible) option is to revalue inventory by posting inventory transactions for all items that are included in the inventory beginning balances.
