General Journal Entry

 

General Journal Entry is a transaction used to create or void accounting transactions.

You can create new General Journal Entries as well as view, print, and save them in electronic form for sending in e-mail.

 

To enter a General Journal Entry:

 

1. Open the General Journal Entry window by choosing Create / General Journal Entry in the Accounting workspace. Alternatively you can create a new General Journal Entry by selecting Create in the General Journal Entries list in the Accounting workspace.

2. Change the default date, if necessary.

3. Enter line items.

4. Save the General Journal Entry. The system will create transactions in the General Journal. Note that system doesn't allow saving unbalanced transactions unless the Unbalanced GL Entry option is turned on (Lists / Settings / General Settings). It's recommended that the only case when unbalanced transactions are posted is for entering beginning balances.
