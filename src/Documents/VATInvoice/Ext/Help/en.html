VAT Invoices

 

VAT Invoice is a document that closes you VAT liability (or negative liability) for the time period and creates invoices to pay the liability (or to record a tax refund). You can create new VAT Invoices as well as view, print, and save them in electronic form for sending in e-mail.

 

To enter a VAT Invoice:

1. Open the VAT Invoice window by selecting Create in the VAT Invoices list in the Accounting workspace.

2. Change the default date and due date, if necessary.

3. Select the collecting (or refunding) agency.

4. Select is this is a payment or a refund.

5. Enter the amount.

5. Save the VAT Invoice.

If the invoice type is Payment the system will create a Vendor Invoice document that can be paid recording a Payment. If the invoice type is Refund the system will create a Sales Invoice document, payment of which can be recording using a Receipt.

The system also generates posting transactions, you can find out detailed information on them in the General Journal section.
