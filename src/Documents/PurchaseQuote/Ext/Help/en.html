Purchase Quotes

 

You can use the Purchase Quote functionality to enter purchase quotes as well as view, print, and save them in electronic form for sending in e-mail.

 

The following documents can be created based on a Purchase Quote:

  • Purchase Order

 

To enter a Purchase Quote:

 

1. Open the Purchase Quote window by choosing Create / Purchase Quote in the Purchasing workspace. Alternatively you can create a new Purchase Quote by selecting Create in the Purchase Quotes list or in the Purchasing Documents journal (both accessible from the Purchasing workspace).

2. Change the default date, if necessary.

3. Select a Vendor. For purchase transactions in the Vendor field you can only select companies of the Vendor type.

If the multi-currency functionality is enabled after you select a vendor its default currency will appear in the currency field on a Purchase Quote and the system will calculate an exchange rate. The default currency and the calculated exchange rate can be changed, if necessary. Also the system recalculates the exchange rate if you change the Purchase Quote date. If you change the currency the system won't recalculate already entered prices in line items, so it's advised to change the currency and the Purchase Quote date before entering line items.

4. If the warehouse locations functionality is enabled select a warehouse in the Location field.

5. Enter line items of a Purchase Quote by selecting Items.

If the units of measure functionality is enabled when you select an Item its default purchase unit of measure (U/M) will appear in the U/M field. After entering the Quantity U/M, the system will recalculate the Base Quantity of the Item. Note, that all inventory tracking in the Inventory Journal is done in the base unit of measure.

If the VAT financial localization functionality is enable the system will automatically calculate VAT based on VAT codes assigned to items. You can change individual VAT codes on items, if necessary.

6. Save the Purchase Quote. The system doesn't generate any General Journal or Inventory Journal postings for Purchase Quotes.

7. You can create a Purchase Order from a Purchase Quote by selecting Create Based On in the Purchase Quote window.
