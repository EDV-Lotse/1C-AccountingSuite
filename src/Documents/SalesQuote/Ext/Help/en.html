Sales Quotes

 

You can use the Sales Quote functionality to enter quotes for prospects and customers as well as view, print, and save them in electronic form for sending in e-mail.

 

The following documents can be created based on a Quote:

  • Sales Order

 

To enter a Sales Quote:

 

1. Open the Sales Quote window by choosing Create / Sales Quote in the Sales workspace. Alternatively you can create a new Sales Quote by selecting Create in the Sales Quotes list or in the Sales Documents journal (both accessible from the Sales workspace).

2. Change the default date, if necessary.

3. Select a Customer. For sales transactions in the Customer field you can only select companies of the Customer type.

If the multi-currency functionality is enabled after you select a customer its default currency will appear in the currency field on a Sales Quote and the system will calculate an exchange rate. The default currency and the calculated exchange rate can be changed, if necessary. Also the system recalculates the exchange rate if you change the quote date. If you change the currency the system won't recalculate already entered prices in line items, so it's advised to change the currency and the Sales Quote date before entering line items.

4. Enter line items of a quote by selecting Items.

If the units of measure functionality is enabled when you select an Item its default sales unit of measure (U/M) will appear in the U/M field. After entering the Quantity U/M, the system will recalculate the Base Quantity of the Item. Note, that all inventory tracking in the Inventory Journal is done in the base unit of measure.

The system will automatically pick a price from the Price List when an Item is selected. An Item price in the Price List is presumed to be set on the default sales unit of measure of the Item in the default currency. If you change the default unit of measure for the line item the system will recalculate the quantity but you'll need to enter a price manually.

5. For the US financial localization functionality is enabled the system automatically calculates sales tax for each Item line, and the total amount for the quote. The sales tax is calculated based on the buyer's sales tax code selected in the Company window. The system automatically selects "taxable" for inventory item types, and "non-taxable" for non-inventory item types (services), which can be changed for each line item if necessary.

If the South Africa financial localization functionality is enabled the system will automatically calculate VAT based on VAT codes assigned to items. You can change individual VAT codes on items, if necessary.

6. Save the Sales Quote. The system doesn't generate any General Journal or Inventory Journal postings for sales quotes.

7. You can create a Sales Order from a sales quote by selecting Create Based On in the Sales Quote window.
