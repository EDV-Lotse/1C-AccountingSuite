Checks

 

The Check is a document used to record payments. There are three business processes where checks can be used:

  • Checks are created based on the Payment and Cash Purchase documents to automate physical check writing.
  • Checks are created manually to automate physical check writing and create transactions recording payments to vendors.
  • Checks are created manually just to create transactions recording payments to vendors.

 

A Check can be created based on:

  • Payment
  • Cash Purchase

 

To enter a Check:

1. Open the Check window by choosing Create / Check in the Banking workspace. Alternatively you can create a new Check by selecting Create in the Checks list in the Banking workspace.

2. The system will autofill the next check number for the selected default bank account. Change the bank account, if necessary. The system won't allow you to save a check with duplicate check numbers for the same bank account.

3. Change the default date, if necessary. The date is used for General Journal posting.

4. Select the Company you would like to pay.

5. In line items select accounts and amounts you would like to pay. Note that line items will be unavailable if the Check is created based on a Payment or Cash Purchase documents.

5. Enter the amounts you would like to pay against each unpaid document.

6. Save the Check. The system will generate General Journal postings. Detailed information about generated posting transactions is provided in the General Journal section.

To print a check click on the Print Check button in the Сheck window. The print form is compatible with the QuickBooks "Check on Top" check format.
