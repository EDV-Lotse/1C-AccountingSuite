Payments

 

The Payment is a document used to record Purchase Invoice and Sales Return payments. You can create new Payments as well as view, print, and save them in electronic form for sending in e-mail.

 

The following document can be created based on a Payment:

  • Check

 

To enter a Payment:

 

1. Open the Payment window by choosing Create / Payment in the Banking workspace. Alternatively you can create a new Payment by selecting Create in the Payments list in the Banking workspace.

2. Change the default date, if necessary. The date is used for General Journal posting.

3. Select the Company you would like to pay.

4. In line items the system will display all unpaid documents for the Company you pay with amounts due.

5. Enter the amounts you would like to pay against each unpaid document.

6. Save the Payment. The system will update Accounts Payable and Accounts Receivable (for Sales Returns) and generate General Journal postings. Detailed information about generated posting transactions is provided in the General Journal section.

7. To create a Check select Create Based On / Check in the check windows. The system will prefill the check with the payment information and suggest the next check number for the selected bank account.
