Deposits

 

The Deposit is a document that records a bank deposit of a payment or several payments that weren't deposited at the time of a receiving payments.  You can create new Deposits as well as view, print, and save them in electronic form for sending in e-mail.

 

To enter a Deposit:

 

1. Open the Deposit window by choosing Create / Deposit in the Banking workspace. Alternatively you can create a new Deposit by selecting Create in the Deposits list in the Banking workspace.

2. Change the default date, if necessary. The date is used for General Journal posting.

3. The system displays in line items all payments that have the "Undeposited funds" tag.

4. Select the bank account that will receive the funds

5. Indicate the undeposited payments included in the Deposit.

6. Save the Deposit. The system will generate General Journal postings. Detailed information about generated posting transactions is provided in the General Journal section.
