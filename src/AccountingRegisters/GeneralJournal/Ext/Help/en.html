General journal

 

The system supports both real-time posting (updating inventory and creating general journal transactions), as well as batch posting. For real-time posting when saving documents use the Post & Close button, for batch posting use the Save button, and later use the Post button selecting the documents you intend to post. Note, that the inventory, A/R and A/P functionality are dependent on General Journal posting, so if a document is not posted it's not reflected in inventory balances, A/R and A/P data.

General journal posting uses accounts set up in three sections - Posting Accounts (located in Lists / General Settings), Companies, and Items. If items in documents are selected from the Items list the system takes item setup accounts. If Items in documents are not selected from the Items list and instead their descriptions are typed in the Description field then the system takes accounts set up on Company (Customer or Vendor) forms.

Note that in the General Journal the system fills in the Company and Document fields only for A/R and A/P accounts. If the multi-currency functionality is turned on the system will record amounts in the Amount FCY (foreign currency) column also only for A/R and A/P accounts.

The system won't allow you to manually modify the General Journal. Use the General Journal Entry document for creating and voiding transactions.

 

The section below describes transactions that are automatically created by individual documents in the system:

 

Purchase Invoice

 

Debit - Inventory or Expense account

Credit - A/P account

Debit - VAT account (for the VAT financial localization)

 

Inventory adjustment amount if adjusted cost higher than actual:

 

Debit - COGS account

Credit - Inventory account

 

Inventory adjustment amount if adjusted cost lower than actual:

 

Debit - Inventory account

Credit - COGS account

 

If the Purchase Invoice is created based on a Goods Receipt then the system won't create any new transactions but will change the date and the posting document for the original Goods Receipt transactions to the Purchase Invoice.
