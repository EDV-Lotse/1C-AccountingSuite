VAT Codes List

 

VAT is the list of value added tax codes your business uses in purchase and sales transactions. The list is available if the South Africa financial localization is turned on.

 

To enter a VAT code:

1. Open the VAT Code window by selecting Create in the VAT Codes list that is accessible from the Accounting workspace.

2. Enter the Code and Description. For example, "S", "Standard VAT".

3. On the Sales Items tab list items that comprise the VAT tax code indicating their name, rate, VAT return form box, and collecting agency.

4. On the Purchase Items tab list items that comprise the VAT tax code indicating their name, rate, VAT return form box, and collecting agency.

5. Save the VAT Code.
