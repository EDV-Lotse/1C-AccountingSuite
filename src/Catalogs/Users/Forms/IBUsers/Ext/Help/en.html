The User accounts form allows you to view the list of all application users. The list includes regular users, with or without access to the infobase.

The list includes the following columns:

  • Full name. The user name, which is taken from the Full name field. Names of users that do not have access to the infobase are displayed in gray.
  • Infobase user name. The name for logging on to the infobase.
  • 1C authentication. Indicates whether the user has the right to log on to the application using their 1C:Enterprise user name and password.
  • OS authentication. Indicates whether the user has the right to log on to the application using their Windows domain user name and password.
  • OS user name. The Windows domain user name.

Double-clicking a user allows you to edit their details.
