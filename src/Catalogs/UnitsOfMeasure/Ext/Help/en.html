Units of Measure List

This list is available if the Units of Measure functional option is turned on. Fill in the list with units of measure you are using in your documents.

For example:

Abbrev.    Description     Parent U/M      Conversion Ratio       Type

dz           dozen            each              12                           Count

ea           each             each               1                            Count

pr            pair              each               2                            Count

 

Note that the Parent (base) unit of measure is entered with itself as the Parent U/M (when entering a new base unit of measure save it first, and then you'll be able to select it in the Parent U/M field).
