The Full Text Search Management form allows you to update and clear the full-text search index.

  • If the application is running in the client/server mode, the index is automatically updated daily at 8:00 AM. In addition to that, you can use the manual update as often as needed.
  • If the application is running in the file mode, you have to update the full-text search index manually.

To update full-text search index

  1. In the Settings section, in the Parameters box, click System settings.
  2. Click Update Index.

To clear the full-text search index

  1. In the Settings section, in the Parameters box, click System settings.
  2. Click Clear Index.
