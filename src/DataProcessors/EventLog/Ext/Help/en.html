The Event Log form allows you to view the customizable event log (all of the actions performed in the application, including user actions and scheduled jobs).

The number of stored log entries is specified by the application administrator.

To view the Event Log

  • In the Settings section, in the left pane, click Event log.

    You can specify the number of latest log entries displayed in the Number of events displayed box. To view earlier entries, filter the log by time.
