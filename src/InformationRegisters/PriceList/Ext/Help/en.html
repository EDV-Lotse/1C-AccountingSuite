Price List

 

The Price list stores daily item prices that are automatically selected into sales documents. Prices are stored in system company's default currency set up in the Settings window accessible in the Lists workspace.

If the Advanced Pricing functionality is enabled (turned on in the Settings window) then the system also stores prices by Customer. When a new Item is added into a sales document if the Customer is selected the system will automatically pull the Customer's price. If the Customer field is empty then the system will select a generic price, if available.
